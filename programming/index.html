Python - replace part of string with other string

Often when dealing with strings you want to replace some part of it with other. Usually, you'd do: 

text = 'The cake is a lie.'

text = text.replace('a lie', 'delicious')

But what if string contains multiple instances of the text you want to replace, and you don't have to replace all of them?

Here is a little snippet that may come in handy: 

def replace(org, rep, start, end):
    """
    Injects rep into org replacing characters from start to end.
    Returns modified string.
    
    org: string to work on
    rep: string that will be injected
    start: index of first char to be erased
    end: index of last char to be erased
    """
    org = org[:start] + rep + org[end:]
    return org

rep will replace characters from start to end in string org. Like this: 

text = 'Spam spam spam, spam spam!'    
print replace(text, 'wonderful', 16, 20)

# Output:
# Spam spam spam, wonderful spam!

The start and end indexes are easy to find, for example using regular expressions, but this is not the topic of this course ;-).
